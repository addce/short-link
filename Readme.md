### How to run backend

- change dir`cd backend`
- install `npm i`
- run test  `npm test`
- start service  `npm start`

### How to run frontend

- install python
- change dir `cd frontend`
- run `python -m http.server 8080`
- you need change `13.58.176.19:3000` to `localhost:8080` in `frontend/index.html`file
